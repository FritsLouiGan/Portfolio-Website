Explaination of Backend and frontend technology/s that i used.

HTML (Hypertext Markup Language) is the standard language used to create and design web pages on the internet. It provides a structure for web content through a system of tags, which define the elements and their layout within a page. By using HTML, web developers can organize text, images, links, and other media to create visually appealing and interactive websites.

Bootstrap is a popular front-end framework used by developers to create responsive and visually appealing websites and web applications. It provides pre-designed templates, CSS styles, and JavaScript components that streamline the development process, allowing for faster and more consistent design implementation. By leveraging Bootstrap's grid system, typography, and UI components, developers can build responsive and mobile-friendly interfaces with ease, enhancing user experience across various devices and screen sizes.

CSS (Cascading Style Sheets) is a language used to style and format HTML documents, controlling the appearance of web pages. It allows developers to define properties such as colors, fonts, margins, and layouts to enhance the visual presentation of content. By separating the structure (HTML) from the design (CSS), developers can create flexible and customizable websites that adapt to different devices and screen sizes, improving accessibility and user experience on the web.

JavaScript is a versatile programming language commonly used in web development to create dynamic and interactive elements on web pages. It enables developers to add functionality such as form validation, interactive maps, animations, and responsive behavior to websites. JavaScript interacts with the HTML and CSS of a webpage, allowing developers to manipulate the content and respond to user actions, ultimately enhancing the user experience and making web applications more engaging and user-friendly.

PHP (Hypertext Preprocessor) is a server-side scripting language primarily used for web development and dynamic content generation. It allows developers to create interactive and dynamic web pages by embedding PHP code within HTML. PHP can interact with databases, handle form data, manage sessions, and perform a wide range of server-side tasks, making it a powerful tool for building dynamic websites and web applications. Additionally, PHP is open-source and supported by a vast community, offering extensive libraries and frameworks to streamline development processes.
